## ADDED Requirements

### Requirement: 扩展后台必须支持抓取二进制媒体资源
系统 MUST 允许扩展后台代表内容脚本抓取远端音频等二进制媒体资源，并将结果以内容脚本可消费的形式返回，而不是要求页面上下文直接访问远端媒体 URL。

#### Scenario: 抓取分段音频资源
- **WHEN** 内容脚本请求扩展后台抓取某个 Provider 分段音频 URL
- **THEN** 扩展后台 MUST 返回可供页面音频播放使用的二进制结果或等价的本地可播放引用

#### Scenario: 抓取最终整段音频资源
- **WHEN** 内容脚本请求扩展后台抓取某个完整中文配音音频 URL
- **THEN** 扩展后台 MUST 返回可供页面音频播放使用的二进制结果或等价的本地可播放引用

### Requirement: 扩展必须提供可释放的媒体引用管理
系统 MUST 为后台抓取得到的媒体资源提供明确的创建、复用和释放机制，以避免 `blob:` 或等价本地引用无限累积。

#### Scenario: 复用已缓存的媒体资源
- **WHEN** 内容脚本再次请求同一个媒体 URL 且其缓存仍在有效窗口内
- **THEN** 扩展 MUST 允许复用已有的本地媒体引用，而不是重复下载同一资源

#### Scenario: 释放过期媒体资源
- **WHEN** 某个媒体资源已经移出当前播放窗口或不再需要
- **THEN** 扩展 MUST 释放对应的本地媒体引用或等价缓存资源
