## ADDED Requirements

### Requirement: 用户可以基于现有中文字幕发起中文配音任务
系统 MUST 在检测到可用中文字幕时，允许用户从扩展面板发起中文配音生成任务，并将字幕内容、时间轴和所选 Provider 配置提交给外部服务。

#### Scenario: 可用中文字幕时显示任务入口
- **WHEN** 扩展检测到当前视频存在可读取的中文字幕轨或用户已导入中文字幕文件
- **THEN** 扩展显示可用的“生成中文配音”任务入口

#### Scenario: 缺少中文字幕时阻止任务发起
- **WHEN** 当前视频不存在可读取的中文字幕，且用户未导入任何中文字幕文件
- **THEN** 扩展 MUST 阻止发起中文配音任务，并向用户提示当前前置条件不满足

### Requirement: 系统可以查询并展示中文配音任务状态
系统 MUST 支持跟踪中文配音任务的状态，并在排队、处理中、成功和失败等阶段向用户展示明确状态。

#### Scenario: 任务处理中
- **WHEN** 用户成功发起中文配音任务且外部服务尚未返回结果
- **THEN** 扩展显示任务进行中状态，并允许后续轮询或刷新结果

#### Scenario: 任务失败
- **WHEN** 外部服务返回任务失败、鉴权失败或模型处理错误
- **THEN** 扩展 MUST 向用户展示可理解的失败状态和错误信息

### Requirement: 用户可以切换到生成后的中文配音播放
系统 MUST 在中文配音任务成功后允许用户启用中文配音，并在播放时将原视频与中文配音音频保持同步。

#### Scenario: 启用中文配音
- **WHEN** 当前视频存在可用的中文配音结果且用户启用中文配音
- **THEN** 扩展 MUST 加载中文配音音频并将其与视频播放状态保持同步

#### Scenario: 切回原声
- **WHEN** 用户关闭中文配音模式
- **THEN** 扩展 MUST 停止外挂中文配音播放并恢复原始视频的默认音频行为

### Requirement: 启用中文配音时仍须保留中文字幕显示
系统 MUST 在中文配音启用期间继续支持中文字幕显示，且不因切换到中文配音而丢失字幕覆盖层。

#### Scenario: 中文配音与中文字幕同时启用
- **WHEN** 用户启用中文配音且当前存在可用中文字幕
- **THEN** 扩展 MUST 继续显示中文字幕，除非用户显式关闭字幕显示

#### Scenario: 中文配音结果返回附带新字幕
- **WHEN** 中文配音任务结果同时返回新的字幕资源
- **THEN** 扩展 MUST 至少保留一种可用中文字幕显示路径，不得出现启用中文配音后无中文字幕可见的状态
